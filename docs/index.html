WhatsApp Web, as a real desktop app

A feature-rich, MIT-licensed desktop client for WhatsApp Web — chat themes, a privacy blur, scheduled messages, multiple accounts, spell-check, an app lock and more.

Get it from the Snap Store Downloads Source on GitHub ♥ Sponsor

Install

On any distribution with snapd:

snap install whatly

Also available as Flatpak, AppImage, Debian/Ubuntu, Fedora/COPR, Arch (AUR) and Windows — see the install guide.

Everything it does

One codebase, a proper desktop app.
